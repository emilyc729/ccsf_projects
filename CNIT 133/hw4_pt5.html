Homework 4 Part 5 - List Format Choices

Create a webpage that contains a script that uses a form text box for one of three different list formats. Use jQuery to show/hide instructions and validate input data. Use the following to determine which format is desired. Your script should use a switch statement to help determine this. 1 - unordered list, 2 - ordered number list, 3 - ordered letter list After determining which list format has been chosen, write an appropriate header line and then use a for statement to write 3 lines in the chosen list format. Finally if the inputted value is NOT 1, 2, or 3 then display an error message using jQuery.
