Homework 6 Part 1 - Using Math methods

Create a webpage that contains a script that has a function that uses Math.round, Math.sqrt and Math.floor. Your page should allow entering a floating-point number with at least 4 decimal positions. Your script should display the original floating-point number, the number rounded to the nearest integer, the square root of the number that is then rounded to an integer, the number rounded to the nearest tenth, the number rounded to the nearest hundredth, and the number rounded to the nearest thousandth. Be sure to use parseFloat() for the inputted number. Use a form input box for user input and a textarea to display the output. When displaying the output, be sure to use toFixed() for displaying the number to the nearest tenths, hundredths, and thousandths postions. Validate the input and display an error message if the input is invalid. It is acceptable to use jQuery for this problem.
