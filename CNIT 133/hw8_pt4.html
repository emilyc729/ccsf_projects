Homework 8 Part 4 - Retrieve State Data (Ajax + PHP)

Create a webpage containing a form and an Ajax script or the jQuery $.get() method that will receive data sent from the server. The form should contain input text boxes for state code, state name, state capital and state population. Use readonly="readonly" for the state name, state capital and state population text boxes. Be sure to include a reset button. After the user enters the state code (lower case or CAPS) in the state code text box and then clicks outside this text box, the Ajax script should access a php script on the server which contains the state code (lower case or CAPS) and the corresponding state name, state capital and state population. The script should either return the corresponding state name, state capital and state population into their respective text boxes or 'state code error' into those boxes.
